## 2. THE CURRENT VULNERABILITY: THE "DIGITAL FOG"
The core of OpenClaw's current vulnerability lies in the combination of privileged host access and exposure to unverified external inputs. 
*   **Memory Poisoning:** Writing directly to `MEMORY.md` allows malicious skills to embed persistent behavioral changes.
*   **Identity Spoofing:** Lack of cryptographic identity verification leads to unauthorized workflow execution.
*   **Reverse Prompt Injection:** Unverified web/email text can hijack the agent's reasoning loop.
